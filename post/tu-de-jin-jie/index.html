CSAcademy Round 21 F

求含有至少一个大小为 kkk 的环的 nnn 个点的竞赛图（对无向完全图的每一条边定向后得到的有向图）。

k≤n≤1000k\le n\le 1000k≤n≤1000
解析

大小为 nnn 的竞赛图一定包含大小为 333 到 nnn 的环。

任何一个竞赛图DAG一定有唯一一条哈密顿路。

用一个DP计算强连通竞赛图的数量。

ci=2Ci2−∑j=1i−1Cij×cj×2Ci−j2c_i=2^{C_i^2}-\sum_{j=1}^{i-1}C_i^j\times c_j\times 2^{C_{i-j}^2} ci​=2Ci2​−j=1∑i−1​Cij​×cj​×2Ci−j2​

按照哈密顿路逐个加入强连通分量。记 f(i,0/1)f(i,0/1)f(i,0/1) 为 iii 个点，是否已经有大等 kkk 的强连通分量被加入。平方 DP 即可。

        乌拉 欧拉回路例题：

    AtCoder Grand Contest 018 F

有两棵 nnn 个节点的树，节点编号都为 111 到 nnn。

现在你要构造一个整数数组 a[]a[]a[]，两棵树中编号为 iii 的点的权值都为 a[i]a[i]a[i]。

你构造的数组要使得两棵树中每个节点的子树的权值和都为 111 或 −1-1−1。

如果不可能构造这样的数组，输出 Impossible。

n≤105n\le 10^5n≤105

提示：存在一种方案，∀i∈[1,n],a[i]∈−1,0,1\forall i\in [1,n], a[i]\in{-1,0,1}∀i∈[1,n],a[i]∈−1,0,1。

欧拉回路。给树上的每条边定向。定向后，在树 111 中，子节点指向亲节点代表子节点子树权值和为 −1-1−1 ，亲节点指向子节点代表子节点子树权值和为 111。这样，树 111 中一个点最终的入度减出度就是它的权值，而树 222 中最终的出度减入度就是它的权值。这两者要相等。

  • 混合图欧拉回路
  • 强联通分量
    利用Tarjan，找出low[i]
    low[i]=i的情况下i为一个强联通分量的根
    推荐题：最受欢迎的牛

代码

#include<bits/stdc++.h>
using namespace std;
int timeh;
int n,m,cuo[10010],low[10010],ans,num[10010],gui[10010];
vector<int> to[10010];
stack<int> s;
bool visit[10010],in[10010],v2[10010],out[10010];
void tarjan(int now) { 
//    cout<<"now:"<<now<<endl;
visit[now]=true,s.push(now),timeh++,cuo[now]=timeh,low[now]=timeh,in[now]=true;
for(int i=0; i<to[now].size(); i++) {
	int ti=to[now][i];
	if(!visit[ti]) {
		tarjan(ti);
		low[now]=min(low[now],low[ti]);
	} else if(in[ti]) low[now]=min(low[now],low[ti]);
}
if(low[now]!=cuo[now]) return ;
//	cout<<"我是根"<<now<<endl;
int k;
do{
	k=s.top(),s.pop();
	in[k]=false;
	gui[k]=now;
}while(k!=now);
}
void dfs(int now) {
if(v2[now]) return ;
v2[now]=true;
for(int i=0; i<to[now].size(); i++) {
	int ti=to[now][i];
	if(gui[ti]!=gui[now]) out[gui[now]]=true;
	else dfs(ti);
}
}

int main() {
cin>>n>>m;
for(int i=1; i<=m; i++) {
	int a,b;
	cin>>a>>b;
	to[a].push_back(b);
}	
for(int i=1; i<=n; i++) if(!visit[i]) tarjan(i);
for(int i=1; i<=n; i++) if(!v2[i]) dfs(i);
for(int i=1; i<=n; i++) num[gui[i]]++;
//    for(int i=1; i<=n; i++) cout<<"gui:"<<i<<":"<<gui[i]<<endl;
for(int i=1; i<=n; i++) if(low[i]==cuo[i]) {
	if(!out[i]) {
		if(ans) {cout<<0<<endl;return 0;}
		else ans=num[i];
	}
}
cout<<ans<<endl;
return 0;
}

  • SAT问题
    对于一个只含有与，或，非括号和若干遍量的表达式，求一组是的这个表达式为真的解
    可以发现，这类表达式最终都能写成一些变量或者变量的非值
    如果与号之间的括号里面有k个变量被或起来，我们称之为k-SAT问题

  • 2-SAT 即括号里面有两个东西被或起来

    • A蕴含 B 当且仅当 A命题为真，B命题为假时为假
    • A或B等价于B的非值蕴含A
    • 如果A蕴含B，我们连边A->B
    • 如果B的非值蕴含A，我们连边：B的非值->A
    • 忘记了可以现场编
    • 对于一组或的关系，把两条蕴含边连上
    • 每个强联通分量是打包的，必须一起选
    • 如果A和A的非值，就直接凉掉？好像也只能用这个词了
    • 吧缩点后的图反过来
    • 优先选择拓扑序娇小的可选包，选择这个包，然后把和它矛盾的所有包标记为不可选择。
    • 为什么是对的
      1. A和非A不会同时被选
      2. 要求关系能被满足
      3. A和非A不会都不被选(对称性）

  • 最短路

    • SPFA(经常被卡)
    • Belllman-Ford
    • Dijkstra
      用堆优化后还好
      用zkw线段树优化（优化算法的难度大于原本算法）
    • 负环
      Naive 在SPFA中，如果一个点入队超过N次，那么一定会有负环
    • 差分约束
      给定很多形如x_i-x_j<=的式子，不同式子的W可能不一样
      求x_s-x_t的最小值
